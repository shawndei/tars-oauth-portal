# content-strategy - Project Memory

## Project Overview
Blog content strategy and editorial calendar system for tech company blog.
Created: 2026-02-13

## Goals & Vision
- Establish consistent content publishing schedule (3x per week)
- Target SEO keywords for developer tools and cloud infrastructure
- Build audience through valuable technical content
- Generate 1000+ monthly organic visitors by Q3 2026

## Key Decisions

### Content Pillars (Decided 2/13)
- **Tutorials** - Step-by-step technical guides
- **Best Practices** - Industry standards and patterns
- **Case Studies** - Real-world implementation stories
- **Thought Leadership** - Future trends and predictions

### Publishing Schedule
- Monday: Tutorial or how-to guide
- Wednesday: Best practices or industry analysis
- Friday: Case study or customer success story

### SEO Strategy
- Focus on long-tail keywords
- Target "how to" queries
- Build backlinks through guest posting
- Optimize for featured snippets

## Architecture/Design Notes

### Content Management System
- Platform: Ghost CMS
- Hosting: DigitalOcean
- CDN: Cloudflare
- Analytics: Google Analytics 4 + Plausible

### Editorial Workflow
1. Pitch submitted to content calendar
2. Research and outline phase
3. First draft (2 days)
4. Technical review (1 day)
5. Editorial review (1 day)
6. SEO optimization
7. Publish and promote

### Tools Stack
- **Writing:** Google Docs, Grammarly
- **Planning:** Notion editorial calendar
- **SEO:** Ahrefs, SEMrush
- **Graphics:** Canva, Figma
- **Code Examples:** GitHub Gists

## Important Findings

### Audience Research (Feb 2026)
- Primary audience: Backend developers (Node.js, Python)
- Secondary: DevOps engineers and cloud architects
- Pain points: Deployment complexity, monitoring, cost optimization
- Preferred content: Code examples > theory

### Content Performance Insights
- Tutorial posts get 3x more engagement than opinion pieces
- Code snippets increase time-on-page by 45%
- Video walkthroughs boost social shares by 2x
- Lists and "Top N" articles perform well in SEO
