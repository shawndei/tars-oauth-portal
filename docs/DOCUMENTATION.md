## API Quick Reference

### Core Functions

```javascript
// Create manager
const manager = new SkillManager();

// Register a skill
manager.register({
  name: 'my-skill',
  constructor: MySkillClass
});

// Execute a skill
const result = await manager.execute('my-skill', input);

// Listen to events
eventBus.on('skill-completed', (data) => {
  console.log('Completed:', data);
});
```
