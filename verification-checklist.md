## Verification Criteria

For each capability to be marked ✅ COMPLETE:
1. Skill directory must exist with proper structure
2. SKILL.md must be comprehensive (>500 words, covers usage, examples, integration)
3. Configuration files must be present where applicable
4. Integration points must be clearly documented
5. Test evidence must be present (examples, logs, or test results)

For ⚙️ IN PROGRESS:
- Directory exists but SKILL.md incomplete
- OR configuration incomplete
- OR integration points not fully documented

For ❌ MISSING:
- No skill directory found
- OR directory empty/placeholder only
- OR no evidence of implementation work
